Program ini merupakan sebuah aplikasi sederhana berbasis Python yang dirancang untuk membantu pengguna dalam mengelola data nilai mahasiswa. Seluruh data disimpan menggunakan struktur list dan dictionary, sehingga setiap mahasiswa memiliki informasi tersendiri berupa NIM, nama, serta nilai tugas, UTS, dan UAS. Program ini dioperasikan melalui menu interaktif pada terminal, dan pengguna dapat memilih berbagai perintah untuk memproses data yang tersedia.

Di dalam program ini, disediakan beberapa fitur utama yang memudahkan pengguna dalam melakukan pengelolaan data. Fitur Lihat Data berfungsi menampilkan seluruh data mahasiswa dalam bentuk tabel yang rapi. Jika belum ada data, program akan menampilkan pesan bahwa data masih kosong. Fitur Tambah Data memungkinkan pengguna memasukkan data baru yang kemudian disimpan ke dalam list data_nilai. Sementara itu, fitur Ubah Data memberikan kesempatan untuk memperbarui informasi mahasiswa berdasarkan NIM, dengan perubahan yang hanya dilakukan pada data yang diinputkan kembali oleh pengguna.

Fitur berikutnya adalah Hapus Data, yang bertugas menghapus satu data mahasiswa setelah pengguna memberikan konfirmasi. Ada pula fitur Cari Data, yang digunakan untuk menemukan informasi mahasiswa berdasarkan NIM dan menampilkan nilai akhir yang dihitung menggunakan bobot nilai: 30% tugas, 35% UTS, dan 35% UAS. Program ini juga menyediakan pilihan Keluar untuk mengakhiri proses ketika pengguna tidak lagi ingin menggunakan aplikasi.

Struktur program disusun melalui beberapa fungsi agar lebih teratur dan mudah dipahami. Fungsi-fungsi tersebut mencakup penampilan menu, penampilan daftar nilai, penambahan data, pengubahan data, penghapusan data, pencarian data, dan fungsi utama main() sebagai pusat alur program. Seluruh proses berjalan dalam loop sehingga pengguna dapat melakukan banyak operasi tanpa harus menjalankan ulang program.

Program ini sederhana namun efektif sebagai latihan dasar Python, khususnya dalam memahami input-output, pengulangan, percabangan, serta penggunaan list dan dictionary. Selain itu, format output berupa tabel membantu pengguna membaca data dengan lebih jelas dan terstruktur. Program ini sangat cocok digunakan untuk tugas pemrograman dasar maupun kebutuhan pengelolaan data sederhana.

Program ini dibuat untuk membantu pengguna mengelola data nilai mahasiswa melalui menu interaktif di terminal. Seluruh proses pengolahan data—mulai dari menambah, menampilkan, mengubah, menghapus hingga mencari informasi mahasiswa—ditangani melalui beberapa fungsi yang saling terhubung dalam satu alur utama program. Data mahasiswa disimpan dalam bentuk list of dictionary sehingga setiap entri memiliki struktur yang teratur dan mudah diakses.

Pada bagian awal, program menyediakan fungsi tampilkan_menu() yang bertugas menampilkan pilihan tindakan yang dapat dilakukan pengguna. Menu ini akan terus tampil selama program berjalan, sehingga pengguna dapat memilih operasi yang diinginkan kapan pun. Ketika pengguna memilih untuk melihat data, fungsi tampilkan_daftar() akan menampilkan daftar seluruh mahasiswa dalam format tabel. Jika belum ada data yang tersimpan, program secara otomatis menunjukkan pesan bahwa data masih kosong.

Fungsi tambah_data() memungkinkan pengguna memasukkan data baru, seperti NIM, nama, nilai tugas, UTS, dan UAS. Data yang dimasukkan kemudian disusun ke dalam dictionary dan ditambahkan ke list utama. Data ini selanjutnya dapat diubah dengan menggunakan fungsi ubah_data(), yang mencari mahasiswa berdasarkan NIM. Jika data ditemukan, pengguna diberi kesempatan untuk memperbarui bagian tertentu dari data tersebut.

Untuk proses penghapusan, fungsi hapus_data() bekerja dengan cara yang mirip. Pengguna diminta memasukkan NIM mahasiswa yang ingin dihapus, kemudian program menampilkan data tersebut untuk memastikan keakuratan sebelum dilakukan konfirmasi penghapusan. Jika pengguna menyetujui, data dihapus dari list.

Selain itu, program juga menyediakan fitur pencarian melalui fungsi cari_data(). Dengan memasukkan NIM tertentu, pengguna dapat melihat informasi mahasiswa secara lengkap, termasuk nilai akhirnya. Nilai akhir tersebut dihitung berdasarkan bobot tugas sebesar 30%, UTS 35%, dan UAS 35%. Perhitungan ini dilakukan langsung ketika data ditampilkan.

Seluruh alur program diatur dalam fungsi main(), yang menjalankan loop tak hingga sampai pengguna memilih opsi keluar. Setiap pilihan menu diproses menggunakan percabangan if-elif, memastikan bahwa hanya fungsi yang sesuai yang dijalankan berdasarkan input pengguna.

Secara keseluruhan, program ini menyajikan mekanisme sederhana namun efektif dalam mengelola data nilai mahasiswa. Pengguna dapat melakukan berbagai operasi dengan cara yang interaktif, sementara struktur fungsi yang terpisah membuat kode mudah dipahami dan dikembangkan lebih lanjut. Program ini sangat cocok sebagai latihan konsep dasar Python, khususnya terkait pengelolaan data, penggunaan list-dictionary, struktur percabangan, serta perulangan.
